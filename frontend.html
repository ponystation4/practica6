Frontend

¿Qué es el frontend?


Es la parte visible, la interfaz de usuario, que se presenta en una aplicación para que el usuario interactúe con la aplicación.


Se compone principalmente de HTML, CSS y JavaScript para crear la sección visual de la página.

Fundamentos de frontend


El frontend generalmente consiste en tecnologías como HTML, CSS y JavaScript.


HTML define la estructura del contenido.


CSS se encarga del diseño.


JavaScript proporciona la interactividad.
